Mẹo Thử Nghiệm Đồng Bộ Cloud:

Hãy sao chép link trang này và mở ở một tab ẩn danh hoặc thiết bị khác. Chọn chế độ Cloud và tạo ghi chú để xem phép màu đồng bộ thời gian thực!

Cấu hình Firebase để dùng chế độ Cloud

Tự động đọc config.json cùng thư mục khi chạy qua HTTP server. Hoặc dán firebaseConfig bên dưới. Bật Anonymous trong Authentication.

  1. Chỉnh config.json → reload trang (ưu tiên cao nhất)
  2. Firebase Console → Authentication → bật Anonymous
  3. Firestore → dán rules từ firestore.rules

Lựa chọn chế độ lưu trữ

Đang hoạt động: Offline Local Storage

Dữ liệu được lưu trong trình duyệt của riêng bạn. Nếu bạn tải lại trang, dữ liệu vẫn giữ nguyên, nhưng nếu mở ở trình duyệt khác hoặc thiết bị khác sẽ không thấy.

Đang hoạt động: Realtime Firebase

Dữ liệu được đẩy trực tiếp lên Firebase Cloud. Mọi thiết bị đang mở trang này sẽ cập nhật dữ liệu tự động ngay lập tức (Realtime Sync).
